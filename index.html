Bienvenue sur le site du port de plaisance Russell !

Grâce à ce site internet, vous pouvez vous connecter et gérer les fonctionnalités du port de plaisance Russell

À partir de cette page, vous pouvez vous connecter ou vous inscrire.

Vous pouvez également afficher la documentation de l'API 'Port de Plaisance Russell

Après vous être connecté, vous aurez accès à votre tableau de bord. Vous pourrez alors :

  • Afficher tous les catways, un catway en particulier, en créer, les mettre à jours et les supprimer.
  • Afficher tous les utilisateurs, un utilisateur en particulier, en créer, les mettre à jours et les supprimer.
  • Afficher toutes les réservations, une réservation en particulier, en créer et les supprimer.
